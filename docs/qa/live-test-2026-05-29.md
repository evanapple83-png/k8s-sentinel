## Pass criteria for Phase 1
1. Agent registers (`tunnel registered`).
2. On a tunnel drop / relay restart it **re-registers cleanly** — no `registration rejected` loop. ← the fix.
3. Relay holds `agents:1`.
4. "Scan now" reaches the agent (not `agent offline`); run goes `complete`; findings ingested.
